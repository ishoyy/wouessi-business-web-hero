# Déploiement OVHcloud — BusinessWebHero (Next.js)
## Installation
```bash
cp .env.example .env
npm install
npm run build
npm run start
```
## Variables d’environnement (minimum)
- NEXT_PUBLIC_SUPPORT_EMAIL
- NEXT_PUBLIC_WHATSAPP_NUMBER
- NEXT_PUBLIC_LINKEDIN / YOUTUBE / X / INSTAGRAM / FACEBOOK
- ADMIN_TOKEN (pour /admin)
